2. 문자열 연산자

( 자바와 비슷)

문자열 사이에 덧셈 기호를 사용하면 문자열을 연결할 수 있다. 이때 덧셈 기호를 문자열 연결 연산자 라고 부릅니다. 문자열 + 문자열 -> 문자열 연산자

3. 상수와 변수


상수 : 항상 같은 수 -> 값을 수정을 할 수 없다.
변수 : 변할 수 있는 수 -> 값이 계속 변할 수 있다.
const : 상수
const 이름 = 값
ex] const pi = 3.141592

let : 변수
let 이름 = 값
ex] let pi = 3.141592

변수의 값을 변경할 때는 변수 이름 뒤에 = 기호를 입력하고 값을 넣어주면 됩니다.
변수 = 값

ex]

let g = 9.8
let m = 10
g * m = 98
특정한 이름의 변수는 한 파일에서 한 번만 선언 할 수 있다. 만약 같은 이름으로 변수를 선언하면 오류를 발생한다. 문자열 입력 prompt (메세지 문자열, 기본 입력 문자열) var
